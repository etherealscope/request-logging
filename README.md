# request-logging
Spring boot starter for request logging

Maven

```
<dependency>
    <groupId>com.etherealscope</groupId>
    <artifactId>request-logging-spring-boot-starter</artifactId>
    <version>1.0.0.RELEASE</version>
</dependency>
```

Gradle
```
implementation("com.etherealscope:request-logging-spring-boot-starter:1.0.0.RELEASE")
```
